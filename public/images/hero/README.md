# Hero Image Directory

## Image Requirements

**File Name:** `whatsapp-phone.png`

**Specifications:**
- Format: PNG or JPG
- Minimum Resolution: 1200x1600px (portrait orientation)
- File Size: < 500KB (compress using tinypng.com)
- Content: Smartphone mockup showing WhatsApp conversation with BuildMonitor

## How to Add Your Image

1. Generate or obtain your hero image (AI-generated smartphone with WhatsApp conversation)
2. Save it as: `whatsapp-phone.png`
3. Place it in this directory: `public/images/hero/whatsapp-phone.png`
4. Compress the image to < 500KB using [tinypng.com](https://tinypng.com)
5. The image will automatically appear on the landing page

## Image Display

The image will be displayed:
- **Mobile**: Full width (max 400px), centered, below the text
- **Desktop**: 500px width, right-aligned, next to the text
- **Animation**: Subtle floating motion on desktop
- **Styling**: Rounded corners, shadow, and drop-shadow effect
