# Stock Images Directory

## Required Images

Please add the following stock images to this directory:

### Stats Bar Images (Section 1):
- `contractors.png` - Image representing construction contractors/workers
- `money.png` - Image representing money/financial tracking
- `expenses.png` - Image representing expense tracking/receipts
- `satisfaction.png` - Image representing customer satisfaction/ratings

### How It Works Images (Section 3):
- `whatsapp-message.png` - Screenshot or illustration of WhatsApp message
- `ai-logging.png` - Illustration showing AI processing/logging
- `dashboard-view.png` - Screenshot of dashboard view

### Dashboard Preview (Section 5):
- `dashboard-preview.png` - Full dashboard screenshot (already referenced)

## Image Specifications:
- Format: PNG or JPG
- Recommended size: 400x400px for icons, 1200x800px for screenshots
- File size: < 200KB per image (compress using tinypng.com)
- Style: Professional, modern, construction/budget themed

## Image Sources:
You can find free stock images at:
- Unsplash.com (search: construction, budget, tracking)
- Pexels.com
- Pixabay.com

Or use AI image generators to create custom illustrations.
